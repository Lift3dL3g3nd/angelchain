# AngelChain

This repository represents the original public articulation of the AngelChain concept and its ethical constraints.

AngelChain is a **research-driven transparency protocol** focused on improving the auditability of charitable donation claims.

It is currently in a **pre-operational design phase**.

---

## What This Repository Is

This repository contains:

- Foundational ethical commitments
- Protocol design principles
- Publicly stated assumptions and constraints

AngelChain does **not** currently:

- Process donations
- Custody funds
- Issue tax receipts
- Operate as a charity

## Design Philosophy

AngelChain prioritizes:

- Auditability over speed
- Correctness over adoption
- Transparency over convenience
- Long-term credibility over short-term growth
